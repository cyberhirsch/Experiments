# Experiments

A collection of creative coding experiments and R&D projects.

## Projects

### Flame
A flame fractal renderer implementation exploring 2D and 3D iterated function systems (IFS).
- `Flame2d.html`: 2D implementations of flame fractals.
- `Flame3d.html`: 3D implementation.
- Reference: Based on the "The Fractal Flame Algorithm" by Scott Draves.

### Gas Giant
An experiment visualizing a gas giant, combining 2D fluid dynamics with sphere rendering.
- `flip_sphere_cubemap.html`: FLIP fluid dynamics simulation mapped onto a 3D sphere using cubemaps to simulate planetary atmospheres.
